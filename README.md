# Eyeloss

Codebase visualization tool for mapping file dependencies and viewing version control history.

## Development

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start application:
   ```bash
   npm run tauri dev
   ```

## Features

- Force-directed visualization of files and modules.
- Version control integration.
- Side-by-side split diff viewer.
- Multi-language parsing support.
- Real-time monitoring of local file changes.
